# Installation Guide

## Prerequisites

- Kintone domain (Free or paid)
- Admin access to manage apps and plugins
- Modern web browser (Chrome, Firefox, Safari, Edge)

## Step 1: Download Plugin

Option A: Download from Release Page
- Go to [GitHub Releases](https://github.com/samir0furukawa/kintone_direct_debit)
- Download `kintone-billing-plugin.zip`

Option B: Build from Source
```bash
git clone https://github.com/samir0furukawa/kintone_direct_debit.git
cd kintone-billing-plugin
npm install
npm run release
